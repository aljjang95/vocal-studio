# HLB 보컬스튜디오 — 제품 구조 전면 개편

## 핵심 변경

정액제 삭제. 취미반(10만원) 신설. 온보딩 강화. AI 커버 티어별 통합.

## 요금제 (확정)

| 플랜 | 가격 | 레슨 방식 | AI 커버 | API 한도 |
|------|------|----------|---------|----------|
| **무료** | 0원 | 상담 1회 + 기본 체험 | 내장 저작권 없는 파일만 | — |
| **취미반** | 100,000원/월 | 자유 곡 실시간 평가 + AI 피드백 (커리큘럼 맛보기 일부 + AI 보컬 지식) | 본인 파일 업로드, 목소리 클론 | 5,000원/월 |
| **발성전문반** | 150,000원/월 | 28단계 HLB 커리큘럼 (5-phase, 채점→해금, 카메라/녹음 체크) | 본인 파일 업로드, 목소리 클론 | 10,000원/월 |
| **유료 피드백** | 50,000원/회 | 선생님 직접 듣고 진단 (추가 구매) | — | — |

### API 한도 설명
- "종합 API" = AI 커버 변환 + Claude 채팅 + 분석 등 모든 외부 API 호출 합산
- 초과 시 해당 월 추가 사용 불가, 다음 달 리셋
- billingStore에서 사용량 추적

## 사용자 흐름

### 1. 첫 방문 → 무료 상담 (강제)

```
랜딩 CTA → /onboarding (강제, 다른 경로 없음)
  Step 1: 녹음 OR 파일 업로드
    - 녹음: 애국가 등 아무 노래 한 소절
    - 업로드: 본인이 부른 노래 파일 (.mp3, .wav, .m4a)
    - 경고 배너: "소음이 많거나 음질이 낮으면 정확한 분석이 어렵습니다"
  Step 2: AI 분석 (4축 긴장 감지 + 피치/음정)
  Step 3: AI(HLB 클론) 진단 결과
    - "어떤 문제가 있는지" 구체적 진단
    - "이 부분을 교정하면 가창이 수월해진다" 방향 제시
    - "교정 없이 계속 부르면 이런 문제가 생길 수 있다" 경고
  Step 4: 요금제 선택 분기
    - "부르면서 자유롭게 배우고 싶다" → 취미반 안내
    - "체계적으로 처음부터 배우고 싶다" → 발성전문반 안내
    - "일단 둘러볼게" → 무료 체험
```

### 2. 취미반 흐름

```
자유 곡 선택/업로드
  → 부르기 (실시간 녹음)
  → 실시간 AI 평가 (피치, 긴장, 발성)
  → AI 피드백 (HLB 커리큘럼 맛보기 + AI 보컬 지식 종합)
  → "발성전문반은 체계적 커리큘럼으로 훨씬 빠른 성장 가능" 업셀
  
AI 커버:
  → 본인 목소리 녹음 → 클론 생성
  → 배우고 싶은 곡 업로드 → 내 목소리로 변환
  → "나도 이렇게 부를 수 있겠구나" 동기부여
```

### 3. 발성전문반 흐름

```
현재 "소리의 길" 시스템 유지
  + 카메라/녹음으로 발성 상태 체크 (추후)
  + 28단계 5-phase 레슨 (왜?→시범→실습→평가→요약)
  + 단계별 채점 → 통과 시 다음 해금
  
AI 커버: 취미반과 동일 + 더 높은 API 한도
```

### 4. 무료 사용자 흐름

```
상담 1회 완료 후:
  → 기본 체험 (제한적)
  → AI 커버: 내장 저작권 없는 파일로만 체험
  → 취미반/발성전문반 가입 유도
```

## 변경 범위

### 삭제
- 정액제 (19,900원/월) 요금 카드 및 관련 로직
- `hobby` 티어 → `hobby` = 취미반으로 재정의

### 신규
- 온보딩: 파일 업로드 옵션 (녹음 외)
- 온보딩: 음질 경고 배너
- 온보딩: 요금제 선택 분기 (StepTransition 확장)
- 취미반 메인 페이지: 자유 곡 실시간 평가
- 취미반→발성전문 업셀 컴포넌트
- AI 커버: 티어별 API 사용량 제한 로직
- billingStore: API 사용량 추적 (월별 리셋)

### 수정
- Pricing 컴포넌트: 3카드 (무료/취미반/발성전문반) + 유료 피드백
- 요금제 가격: 취미반 10만원, 발성전문반 15만원
- Nav: 취미반/발성전문반 분기 반영
- 온보딩 결과 → 요금제 선택으로 연결

### 유지
- 소리의 길 28단계 시스템 (발성전문반)
- 5-phase 레슨 흐름
- 백엔드 긴장 감지 엔진
- AI 코치 채팅
- 유료 피드백

## 버그 수정

### 이메일 미인증 시 사이트 접근 차단
- Supabase `getUser()`가 미인증 사용자도 반환하는 문제
- middleware에서 `user.email_confirmed_at` 체크 추가
- 미인증 시 "이메일 인증을 완료해주세요" 안내 페이지로 리다이렉트

## 디자인 방향

- awesome-design 스킬로 최신 트렌드 리서치 → UX 감동 설계
- 온보딩 상담 경험이 "가입하고 싶다" 느낌을 주도록 프리미엄 UX
- 진단 결과 시각화에 집중 (4축 레이더 차트, 파형 비주얼 등)

## 기술 참고

- UserTier: `'free' | 'hobby' | 'pro' | 'teacher'` 유지
  - `hobby` = 취미반 (기존 정액제 대체)
  - `pro` = 발성전문반 (동일)
- API 사용량: Supabase `api_usage` 테이블 또는 profiles JSONB
- AI 커버 변환: 기존 `/api/ai-cover` 엔드포인트 활용
