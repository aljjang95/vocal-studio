# HLB 보컬스튜디오 — 전면 개편 계획

## 핵심 컨셉

**선생님(HLB)의 AI 아바타가 온라인 레슨생을 가르치는 서비스.**
오프라인 레슨 흐름을 그대로 온라인에 옮긴다.

기타 기능(AI 커버, 곡 연습 등)은 커뮤니티 활성화 수단.

## 오프라인 레슨 흐름 (실제, 유튜브 영상 기반)

```
상담 (첫 방문)
  → 레슨생 노래 들어보기 (실력 파악)
  → 문제점 진단 (후두 긴장, 고음 밀어내기 등)
  → 디테일한 설명 ("왜 이런 문제가 생기는지")
  → 개선 방향 제시 ("이런 순서로 나아갈 거예요")
  → 첫 레슨 일정 잡기

첫 레슨 (1회차)
  → "왜" 이 연습을 하는지 설명 + 이해시키기
  → 선생님 시범
  → 학생 따라하기
  → 즉시 피드백 ("그렇지!", "목에 힘 빼봐")
  → 스케일 피아노로 반복 연습
  → 채점/평가 → 다음 단계 결정

이후 레슨 (2주차~)
  → 이전 회차 복습
  → 새로운 개념 도입 ("왜" 먼저)
  → 시범 → 따라하기 → 피드백 → 반복
  → 진도에 따라 가창 적용
```

## 유튜브 분석 기반 레슨 순서 (보컬커리큘럼 프로젝트)

### 1회차 (이천희 1회차 영상 기반)
```
Level 1: 설근 안정화 → 후두 안정 → 허밍
Level 2: 라운딩 → 성대 유연성 → 두성 밸런스 → 순어
Level 3: 성구전환 → 선택적 주의력
```

### 2주차 (김성민 2주차 영상 기반)
```
Level 1: 복식호흡
Level 2: 와인드업 → 평형 상태 → 성대 세팅 → IAO 연결 → 두성 밸런스 → 스케일
Level 3: 1.5 스케일 → '구' 발성 → '마' 발성
```

### 핵심 교수법 (coaching_feedback_patterns.md 분석)
- **감각 기반**: "느낌으로", "감각을 찾아봐" (이론이 아닌 체감)
- **즉시 피드백**: "그렇지!" 230회, "좋아" 196회
- **부정→긍정**: "목에 힘 주지 마" → "목에 힘 빼는 감각을 느껴봐"
- **왜 먼저**: 매 개념마다 "왜 이걸 하는지" 설명 후 실습
- **단계적 접근**: 허밍 → 입 열기 → 올라가기 → 가사 붙이기

## 앱 설계 (개편 후)

### 사용자 흐름

```
1. 첫 방문 → "상담" (온보딩)
   ├─ 노래 한 소절 녹음 (또는 업로드)
   ├─ AI가 4축 분석 (후두/혀뿌리/턱/성구전환)
   ├─ AI가 문제점 상세 설명 (선생님 말투로)
   ├─ AI가 개선 로드맵 제시 ("당신은 여기서 시작합니다")
   └─ → 첫 레슨으로 이동

2. 레슨 (핵심 기능)
   ├─ AI가 "왜" 이 연습을 하는지 설명 (텍스트 + TTS 음성)
   ├─ 시범 영상 또는 오디오 재생
   ├─ 피아노 스케일 자동 재생 → 학생 따라하기 → 녹음
   ├─ AI가 실시간 피드백 ("그렇지!", "목에 힘 빼봐")
   ├─ 채점 → 통과/재시도
   └─ 통과 시 다음 레슨 해금

3. 대시보드 (홈)
   ├─ 현재 레슨 진도
   ├─ 오늘 할 연습
   ├─ 성장 그래프
   └─ AI 코치 채팅 (FAQ + Haiku)

4. 부가 기능 (커뮤니티 활성화)
   ├─ AI 커버 (내 목소리로 변환)
   ├─ 곡 연습 (업로드 → 구간 분석)
   └─ 호흡 트레이너
```

### 레슨 상세 흐름

```
레슨 1개 = 다음 구조:

[1] 왜? (Explanation)
    AI가 이 연습의 목적/원리를 설명
    "설근이 안정되면 후두가 자연스럽게 내려옵니다. 
     남성 100명 중 90명이 여기서 힘을 주거든요."
    → TTS로 음성 재생 (선생님 톤)

[2] 시범 (Demonstration)
    선생님 시범 오디오/영상
    "이렇게 '어~' 소리를 내면서 혀를 편하게 떨어뜨리세요"

[3] 실습 (Practice)
    피아노 스케일 자동 재생
    학생 녹음 (WebSocket 실시간 분석)
    AI 실시간 피드백 표시

[4] 평가 (Evaluation)
    3단계 채점 (이완/음정/종합)
    AI가 결과 설명
    통과 → 다음 레슨 해금
    불통과 → 재시도 + 구체적 개선 포인트

[5] 요약 (Summary)
    이번 레슨에서 배운 것
    다음 레슨 예고
    성장 기록 저장
```

## 요금제 (확정)

| 플랜 | 가격 | 범위 |
|------|------|------|
| 무료 | 0원 (광고) | 1~18레슨 채점 진행, 스케일 자율, 내장 변환 |
| 정액제 | 19,900원/월 | 광고 제거 + 모든 컨텐츠 자유이용 (레슨 해금 제외) |
| 발성전문반 | 150,000원/월 | 28레슨 채점 해금 + 4축 분석 + 매커니즘 해설 + 리포트 |
| 유료 피드백 | 50,000원/회 | 선생님 직접 듣고 진단 (추가 구매) |
